# FrontEnd

¿Que es Shtal? 
El sistema está diseñado como una solución tecnológica para administrar el flujo de herramientas dentro de una organización. Su función principal consiste en registrar y controlar los movimientos de entrada y salida de herramentales, utilizando procesos automatizados para la captura de información.
